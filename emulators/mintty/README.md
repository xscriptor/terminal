## Files

<ul>
  <li><code>themes/*.minttyrc</code>: Color scheme files for Mintty terminal (Git Bash, Cygwin, MSYS2).</li>
</ul>

## Installation

<p>Copy the desired theme to <code>~/.minttyrc</code>:</p>

<pre><code>cp themes/x.minttyrc ~/.minttyrc</code></pre>

<p>Restart Mintty to apply the theme.</p>

<p>To switch themes, copy the corresponding file over <code>~/.minttyrc</code> and restart the terminal.</p>
