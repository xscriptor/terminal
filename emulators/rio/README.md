## Files

<ul>
  <li><code>themes/*.toml</code>: Theme files ready to use with Rio.</li>
</ul>

## Installation

<p>Copy the themes to your Rio themes directory:</p>

<pre><code>mkdir -p ~/.config/rio/themes
cp themes/*.toml ~/.config/rio/themes/</code></pre>

<p>Then set the theme in <code>~/.config/rio/config.toml</code>:</p>

<pre><code>theme = "x"</code></pre>

<p>Reload Rio for the changes to take effect.</p>
