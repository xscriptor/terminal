## Installation

<p>Wave Terminal uses a single <code>termthemes.json</code> file located at <code>~/.waveterm/termthemes.json</code>.</p>

<p>Merge the desired theme into your existing <code>termthemes.json</code>. Example structure:</p>

<pre><code>{
  "x": {
    "display:name": "Xscriptor x",
    "display:order": 1,
    "background": "#050505",
    "foreground": "#f7f1ff",
    "black": "#0a0a0a",
    "red": "#fc618d",
    ...
  }
}</code></pre>

<p>After adding the theme, select it from the terminal block header right-click menu, or set it in <code>settings.json</code>:</p>

<pre><code>"term:theme": "x"</code></pre>
