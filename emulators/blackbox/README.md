## Files

<ul>
  <li><code>themes/*.json</code>: Color scheme files for Black Box terminal (Tilix-compatible format).</li>
</ul>

## Installation

<p>Copy the themes to the Black Box schemes directory:</p>

<pre><code>mkdir -p ~/.local/share/blackbox/schemes
cp themes/*.json ~/.local/share/blackbox/schemes/</code></pre>

<p>Restart Black Box and select the desired scheme from the preferences menu.</p>
