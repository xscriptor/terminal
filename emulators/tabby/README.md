## Files

<ul>
  <li><code>themes/*.xresources</code>: Color scheme files in Xresources format for Tabby.</li>
</ul>

## Installation

<p>Tabby supports Xresources color schemes via the <strong>tabby-community-color-schemes</strong> plugin.</p>

<ol>
  <li>Copy the desired <code>.xresources</code> file to your Tabby color schemes directory:
    <pre><code>cp themes/x.xresources ~/.config/tabby/schemes/</code></pre>
  </li>
  <li>Open Tabby settings → Color Schemes → select <strong>Xscriptor x</strong>.</li>
</ol>

<p>Alternatively, install the <strong>tabby-community-color-schemes</strong> plugin from the Tabby plugin marketplace and place the files in the corresponding directory.</p>
