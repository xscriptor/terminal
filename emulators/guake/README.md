## Files

<ul>
  <li><code>themes/*.xml</code>: Color scheme files for Guake terminal, importable via the GUI.</li>
</ul>

## Installation

<p>Open Guake → Preferences (<kbd>F12</kbd> then right-click → Preferences) → Colors tab → Import.</p>

<p>Select the desired <code>.xml</code> theme file from the <code>themes/</code> directory.</p>

<p>Alternatively, copy the files manually:</p>

<pre><code>cp themes/x.xml ~/.config/guake/schemes/</code></pre>

<p>Then select it from the Colors dropdown in Preferences.</p>
